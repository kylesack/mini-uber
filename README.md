# mini-uber

## Description
This repository provides a tutorial and examples for building a mini version of Uber.

## Features
- User registration and authentication
- Ride request and management
- Admin dashboard

## Technology Stack
- Python
- Flask
- SQLite

## Usage
To run the application, clone the repository and execute:

```bash
pip install -r requirements.txt
python app.py
```
